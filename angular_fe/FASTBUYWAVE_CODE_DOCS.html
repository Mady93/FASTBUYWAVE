Overview

FastBuyWave

A full-featured e-commerce marketplace where users post, browse and respond to product advertisements across multiple categories. Supports USER and ADMIN roles, lazy-loaded category views and an integrated contact / appointment system.

Angular 18 Standalone SCSS + _tokens.scss Google OAuth OpenLayers Maps Angular Material Custom Library (my-lib-inside) Lazy Loading via ngComponentOutlet

Architecture

How the application layers interact at runtime.

💡
DashboardComponent is the universal shell for all authenticated routes. The router passes a category + source data object; RouteManagerService maps this to the correct lazy component which is then rendered via ngComponentOutlet.
